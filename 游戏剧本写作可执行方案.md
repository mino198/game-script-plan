# 游戏剧本写作可执行方案

> 基于「山音超级编剧大师」Skill（GitHub ⭐462）+ 游戏叙事最佳实践  
> 创建时间：2026-06-19

---

## 一、方案概述

游戏剧本与影视剧本的核心差异：**玩家是主动的叙事参与者，而非被动观众**。这意味着所有的戏剧结构都需要为"可互动性"留出空间。

本方案将山音编剧大师的八步流程适配到游戏剧本场景，分为三个阶段、九个步骤，每步都有明确的可交付物。

---

## 二、前置问题（开始前必须回答）

在进入正式创作之前，请先确认以下配置：

### 2.1 游戏类型选择

| 类型 | 叙事结构 | 剧本重心 |
|------|---------|---------|
| **RPG/JRPG** | 主线+支线+角色剧情 | 世界观宏大、角色弧光完整 |
| **视觉小说/Galgame** | 多分支+多结局 | 选择节点设计、情感线细腻 |
| **动作冒险** | 关卡式线性叙事 | 开场钩子强、每关独立精彩 |
| **独立叙事游戏** | 步行模拟/环境叙事 | 世界观暗示、物件叙事 |
| **MMORPG** | 主线+世界事件+玩家驱动 | 模块化、可扩展 |

### 2.2 叙事体量

- [ ] **短篇**：1-3小时游玩，单一核心故事
- [ ] **中篇**：5-10小时，主线+2-3条支线
- [ ] **长篇**：20小时+，多章节、季度弧线

### 2.3 互动深度

- [ ] **线性叙事**：玩家无重大选择，故事单一走向
- [ ] **分支叙事**：玩家选择影响剧情走向（2-5个节点）
- [ ] **非线性叙事**：玩家可自由探索，故事碎片化重组

## 三、九步可执行流程

### ▶ STEP 1：世界观架构（对应山音 Step 4 前史与世界观）

游戏剧本必须先建世界，再写故事。

**交付物：世界观圣经（Game Bible）**

```
【世界设定】
  背景时代/世界：
  核心规则（这个世界最重要的1-3条运行法则）：
  权力结构（谁掌握什么资源）：
  世界的主要矛盾（玩家最终要面对的根本冲突）：

【三层世界观】
  表层规则（玩家在序章看到的）：
  运行逻辑（中期揭示的深层机制）：
  底层真相（结局前才揭示的真相）：

【信息释放计划】
  第几章揭示什么层：
  通过什么方式揭示（NPC对话/环境线索/剧情CG）：
```

---

### ▶ STEP 2：核心戏剧动作（对应山音 Step 1 破题）

**公式：** 主角 + 追求什么（Want） + 被什么阻碍（Conflict） + 为什么重要（Stakes）

**游戏版扩展：**
```
主角的 Want（玩家代理的目标）：
主角的 Need（角色真正需要的，玩家要帮他发现）：
核心阻碍（反派/体制/自身弱点）：
Stakes（如果失败，什么会消失）：
玩家的核心体验（让玩家感到什么情感）：
```

**游戏剧本特有要求：**
- Want 必须可以转化为具体的游戏目标（任务/关卡/选择）
- Stakes 必须是玩家能感知的——不能只是"世界会毁灭"，要是"这个玩家已经喜欢上的角色会死"

---

### ▶ STEP 3：主角与主要角色设计（对应山音 Step 3 人物深度）

**主角卡（Ghost/Lie/Flaw/Want/Need 完整链条）**

```
【主角】
  姓名/代号：
  Ghost（前史创伤，在游戏开始之前发生的）：
  Lie（主角相信的谎言，驱动错误的Want）：
  Flaw（Lie在行为层面的外化，玩家一开始会看到的缺陷）：
  Want（主角主动追求的，可转化为游戏目标）：
  Need（主角真正需要的，游戏结尾才实现）：
  弧光：从 [A状态] → 到 [B状态]

【主要配角（至少3个）】
  角色1：功能（镜像/催化/主题/节奏） + Want + 矛盾特征
  角色2：...
  角色3：...

【反派/核心阻碍】
  反派的 Want（他/她为什么做这件事，必须合理）：
  反派的 Lie（反派相信的谎言，让反派觉得自己是对的）：
  反派与主角的镜像关系（两人的本质相似之处）：
```

**游戏剧本特有要求：**
- 配角必须有自己的游戏功能（队友/商人/任务发布者），但不能只是功能性NPC
- 反派必须有令人信服的动机——玩家能理解但不认同

---

### ▶ STEP 4：章节结构大纲（对应山音 Step 5 结构大纲）

按游戏体量选择结构：

#### 短篇/独立游戏（推荐：四幕结构）

```
第一幕：建立（约15%时长）
  开场钩子（前5分钟必须让玩家"想继续玩"）：
  常态展示（玩家的日常、世界的规则）：
  激励事件（打破平衡的事件）：

第二幕前半：适应（约25%时长）
  玩家进入"新规则"世界：
  Fun & Games（玩家探索新系统的乐趣）：
  第一个重大转折（伪胜利/伪失败）：

第二幕后半：危机（约35%时长）
  敌人/困难升级：
  玩家做出代价最大的选择：
  一切尽失时刻（最低点）：

第三幕：决战与落幕（约25%时长）
  主角带着新认知面对最终挑战：
  高潮选择（玩家最终的核心选择）：
  结局与新常态：
```

#### 长篇RPG（推荐：章节制 + 季度弧线）

```
【季度弧线】
  主线核心冲突：
  分几章/几个大节点：
  每章的情绪基调：（建置/轻快/紧张/呼吸/崩塌/高潮）
  弧光消耗规划：前1/3角色变化不超过15%

【单章结构】（每章独立精彩）
  章节Logline：
  开场钩子：
  本章核心戏剧动作：
  本章对主线的推进：
  章末钩子（引导玩家继续）：
```

### ▶ STEP 5：分支节点设计（游戏独有）

这是游戏剧本与影视剧本最大的差异所在。

**分支节点设计原则：**
1. 选择必须有意义——每个选项都应该是"真的可以选"，而非明显一优一劣
2. 选择反映角色的 Lie vs Need——选A代表继续坚持Lie，选B代表开始走向Need
3. 选择有短期后果和长期后果
4. 避免"选择幻觉"——不要让两个选项最终导向完全相同的结果

**分支节点模板**

```
【节点 #X：节点名称】
  触发时机（什么情况下玩家面对这个选择）：
  核心冲突（玩家在什么价值之间做选择）：

  选项A：[选项文本]
    代表的价值取向：
    短期后果（接下来1-2个场景发生什么）：
    长期影响（对结局的影响）：
    角色弧光影响（走向Lie还是Need）：

  选项B：[选项文本]
    代表的价值取向：
    短期后果：
    长期影响：
    角色弧光影响：

  （可选）选项C：隐藏选项，需要特定条件才能出现
```

**分支管理建议：**
- 用树状图可视化全部分支
- 建议节点数量：短篇3-5个，中篇8-12个，长篇15-25个
- 关键分支（影响结局）不超过3个，避免结局太多难以保证质量

### ▶ STEP 7：对话系统设计（游戏独有）

游戏的 NPC 对话和影视台词有根本区别：玩家可以反复触发、可以跳过、可以乱序阅读。

**对话设计原则：**
1. **第一句话决定一切**——玩家可能只看第一句就按跳过，必须让第一句有吸引力
2. **每段对话独立成立**——不能假设玩家记得上一段对话
3. **台词有记忆点**——主要角色的台词应该有辨识度的说话风格
4. **环境叙事优先**——能用场景/物件表达的，不用NPC说出来

**对话类型与规范**

| 类型 | 特点 | 注意事项 |
|------|------|---------|
| **剧情对话（强制）** | 玩家必须看完，不可跳过 | 不超过3分钟，信息量控制 |
| **NPC日常对话** | 玩家主动触发，可跳过 | 每次触发内容要有变化（至少3条） |
| **分支对话** | 玩家选择，影响后续 | 见 Step 5 分支节点设计 |
| **内心独白** | 主角的内心声音 | 稀少使用，不能替代画面 |
| **环境叙事** | 物件/场景传递信息 | 优先于NPC对话 |

---

### ▶ STEP 8：伏笔系统（对应山音伏笔登记表）

游戏的伏笔比影视更强大，因为玩家会主动探索。

**伏笔登记表**

```
| 伏笔编号 | 伏笔内容 | 首次出现（场景#X） | 表面含义 | 回扣位置（场景#Y） | 真实含义 | 已实现 |
|---------|---------|-----------------|---------|-----------------|---------|--------|
| F001    |         |                 |         |                 |         | ❌     |
```

**游戏伏笔分级：**
- **A级（核心）**：与结局直接相关，必须回扣
- **B级（重要）**：丰富世界观和角色，建议回扣
- **C级（彩蛋）**：给仔细探索的玩家准备，可不回扣

### ▶ STEP 9：剧本医生（对应山音 Step 8 剧本医生）

**通用检查清单**

```
【叙事层面】
- [ ] 主角的 Want 是否转化为了清晰的游戏目标？
- [ ] 反派的动机是否令人信服？
- [ ] 玩家在游戏中途是否会对角色产生情感依附？
- [ ] 结局是否有情感分量？

【结构层面】
- [ ] 开场5分钟是否成功建立了"想继续玩"的欲望？
- [ ] 是否有"一切尽失"的最低点？
- [ ] 是否有足够的"呼吸"段落让玩家休息？
- [ ] 节奏是否有起伏，不是一路高强度？

【游戏层面】
- [ ] 剧情推进是否与游戏玩法自然融合（而非割裂的过场动画）？
- [ ] 分支节点的选择是否真的有意义？
- [ ] 玩家的选择是否得到了可感知的回应？
- [ ] 关键剧情是否可以被玩家"遗漏"（如果可以，是否合理）？

【对话层面】
- [ ] 主要角色的台词是否有辨识度？
- [ ] 是否有过于直白的"解释设定"台词？
- [ ] NPC日常对话是否有变化，不是一直说同一句话？

【伏笔层面】
- [ ] 所有A级伏笔是否都有回扣？
- [ ] 回扣是否让玩家产生"原来如此"的感觉？
```
