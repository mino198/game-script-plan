# 《梦鬼》剧本 · 时间轴B
## 外层Meta流程 + 七个反转节点
> 版本：v0.1 | 最后更新：2026-06-19
> 说明：外层是玩家在「2026年现实视角」下的体验层，通过Win98桌面环境与内层游戏交织推进。

---

## 一、外层基础设定

**玩家身份**：林静（林智子的2026年新身份）
**职业**：二手物品收藏/小店经营者
**获得电脑的方式**：在一次线上竞拍中偶然拍到，卖家账号已注销，无法联系

**外层界面形式**：
- 完整模拟Windows 98桌面
- 玩家的「行动」体现为：点击文件、查看文件夹、尝试运行程序
- 外层与内层的信息相互渗透，每完成一个内层章节，外层桌面会发生变化

---

## 二、外层时间线（与内层对应）

| 外层节点 | 对应内层进度 | 外层发生的事 |
|---------|------------|------------|
| 开机 | 游戏启动前 | 发现桌面，读取日记文件夹 |
| 节点① | Chapter 1结束 | 无明显变化，但桌面壁纸颜色轻微偏移 |
| 节点② | Chapter 2结束 | 桌面新增`0920.txt`（陈默管道声音） |
| 节点③ | Chapter 3结束 | `0920.txt`部分损坏内容恢复 |
| 节点④ | Chapter 4结束 | 存档报错，SLOT 3出现 |
| 节点⑤ | Chapter 5开始前 | 桌面图标开始轻微错位 |
| 节点⑥ | 结局分支触发时 | 根据结局不同，桌面出现不同变化 |
| 终幕 | 结局完成后 | 镜子演出，全剧终 |

## 三、七个递进反转节点

（反转核心：玩家=林智子。以下节点从轻微违和到强烈震撼，梯度分布。）

---

### 反转节点① — 旋律的熟悉感
**位置**：Chapter 1，副本一，触摸耳机选项

**触发内容**：
智子拿起耳机，听到一段极低音量的旋律。
智子说「我好像听过，但想不起来」。

**外层对应**：
玩家此后进入内层游戏时，这段旋律会作为背景音效极其低频地循环——
低到玩家不确定是否真的听到了，以为是环境音或者错觉。

**反转揭示时**：
结局A尾声，玩家（林静）在真实房间里「无缘无故地哼出了这段旋律」——她一直知道它，只是从来没意识到。

**情绪意图**：最初的微弱钩子。「我好像听过」的轻微违和。

---

### 反转节点② — 夏冰的那句话
**位置**：Chapter 4，夏冰离开前的最后台词

**夏冰的最后一句话**：
「你是这里面唯一一个，不是为了自己的愿望来的。」
「我觉得……这种人，应该赢。」

**外层对应**：
Chapter 4结束后，外层桌面`0918.txt`（日记文件）内容被更新，最后多了一行：
```
今天在学校，有个同学突然对我说：
"你这种人，应该赢。"
我不知道她在说什么，她也说不清楚。
但我记住这句话了。
```

**反转揭示时**：
玩家（林静）在现实中收到一条很久以前的旧消息/便条，上面写着这句话——不是夏冰写的，但是同一句话，同一个字序。
林静在日常生活中被这句话触动过，只是从来不知道为什么。

**情绪意图**：「我在游戏里听过这句话」→「玩家在游戏外也见过这句话」。两条线开始靠近。

---

### 反转节点③ — 机关的直觉
**位置**：Chapter 3，副本三机关解谜

**触发内容**：
如果玩家在副本一探索阶段没有仔细查看耳机，密码箱谜题应该无法直接解答，需要寻找补充线索。

但有一个隐藏情况——
即使玩家没有在副本一找到任何线索，第一次面对密码箱时，如果玩家尝试输入`3-16-88`，它会直接打开。

**对话触发**（输入正确密码时，不论是否找过线索）：

*（智子）*
*我……怎么知道这个密码。*
*我没有在这里找到任何数字。*
*我只是——想到了这串数字，然后输进去了。*
*……*
*这是什么。*

**情绪意图**：程序性记忆的第一次明确显现。「你/智子知道一件不该知道的事」。这是反转的机制锚点。

---

### 反转节点④ — 「0920.txt」的第二轮更新
**位置**：Chapter 3结束，外层桌面

**外层变化**：
`0920.txt`部分损坏内容修复，出现这段文字：
```
他的声音和我记得的不一样。
更远。
像是从墙壁后面传过来的。
我以为我进来是为了带他出去，
但现在我不确定
他还在不在这里。
```

**关键句**：「我以为我进来是为了带他出去」

**外层玩家（林静）的反应**：
玩家此时读到这句话，会感到一种奇怪的共鸣——
不是「我理解这种心情」的共鸣，
而是「我好像……也有过这种感觉」的违和共鸣。

**外层随机触发**（约30%概率，触发后无法再次触发）：
玩家点击桌面任意位置时，出现一个系统弹窗：

```
┌────────────────────────────┐
│  提示                         │
│  你似乎在找什么。               │
│  你知道自己在找什么吗？           │
│  [是]    [否]    [我不确定]    │
└────────────────────────────┘
```

无论选哪个，弹窗关闭，不做任何额外反应。
但这个问题会在玩家脑中留下残影。

**情绪意图**：游戏开始直接「问」玩家。边界开始模糊。

---

### 反转节点⑤ — 陈默认出了她
**位置**：Chapter 4，镜廊，陈默的倒影

**触发内容**（已在时间轴A剧本中写出）：
陈默的倒影叫出了「智子」的名字——
但在这个游戏场景里，智子没有向他做过自我介绍。

**关键台词**：
「你来了。」（副本一他说的）
「你知道我叫什么吗。」（智子没有问出口，但玩家感受到了这个疑问）

**外层对应**：
Chapter 4结束后，外层桌面出现一个新文件：
`照片.bmp`（.bmp格式，老式位图图片）

打开图片——
一张颗粒感很重的老照片，是两个高中生，教室背景。
照片里的两个人：一个面孔模糊（陈默），一个面孔清晰——

那张清晰的脸，和玩家屏幕里倒映的自己的脸，非常像。

但照片的时间戳是：`2000-09-12`。

*（旁白·林静）*
*这张照片……*
*这个人……*
*这不可能是我。*
*这是2000年的照片，那时候我才——*

*（停顿）*

*……我几岁。*
*我什么时候出生的。*

（这是玩家第一次意识到：她不太确定自己的年龄。）

**情绪意图**：照片与玩家的脸重叠。认知开始动摇。

---

### 反转节点⑥ — LAST.txt 的解锁
**位置**：结局分支触发前（Chapter 5进行中）

**触发条件**：玩家在外层桌面上再次点击之前损坏的`LAST.txt`

**这次，文件打开了。**

内容：
```
我赢了。

但我好像忘了什么。

我记得我还有个愿望——
不是那个我告诉别人的愿望。
是那个我自己也说不出口的愿望。

是关于他的。

他叫什么名字，我已经不记得了。
但我记得他说过的最后一句话：
「没关系的，智子。」

我赢了，但这句话还留在我这里。
它没有被清洗掉。
因为它在我的某个地方，
比记忆更深。

如果有一天，有人再次看到这个文件——
那个人可能就是我。

——林智子，2000年9月
```

*（旁白·林静）*
*「那个人可能就是我。」*

*……*

*我的名字是林静。*
*不是林智子。*
*我叫林静。*

*（停顿）*

*……对吗。*

**情绪意图**：玩家第一次正面质疑自己的名字。反转的引信点燃。

---

### 反转节点⑦ — 镜子
**位置**：任意结局的最后一幕（结局A尾声）

**外层演出**：
游戏结束，屏幕变暗。
旁白提示：「我拿起桌上的镜子。」

（此时游戏界面变黑，只有摄像头授权弹窗出现——）

```
┌───────────────────────────────────────┐
│  梦鬼.exe 请求使用摄像头                   │
│  这将允许游戏访问你的摄像头画面。             │
│  [允许]        [拒绝]                   │
└───────────────────────────────────────┘
```

**【若玩家允许】**：
屏幕显示摄像头画面——玩家自己的脸，实时的。
画面旁边，半透明叠加显示游戏里林智子的黑白肖像照。

两张脸并排。
游戏不做任何判断，不说「你们一样」，不说「你就是她」。
只是并排放在那里。
等待玩家自己看到。

旁白（极小字）：
`「没关系的，智子。」`

然后屏幕黑掉。

---

**【若玩家拒绝】**：
屏幕继续黑着。
只有一行文字：

`「没关系。你本来就知道了。」`

---

**情绪意图**：这是整个游戏最核心的一刻。
不是游戏告诉你「你就是林智子」，
而是游戏把你和她并排放在一起，让你自己去感受那个重叠。
Meta的终点不是震惊，而是——一种说不清楚的、安静的认出。

## 五、外层隐藏细节（彩蛋层）

**细节①：桌面壁纸**
Win98默认壁纸是蓝绿色渐变。
但仔细看右下角，有一个极小的、几乎看不见的图案——
是两个并排站立的人形剪影。
第一次看不会注意，但结局A后再回来看，它会变成单独一个人形。

**细节②：任务栏时间**
游戏开始时，桌面时间是`2000年9月14日 23:47`。
每次进入/退出内层游戏，时间会偶尔「闪动」一下，变成`2026年XX月XX日`——当天的真实日期。
然后变回2000年。

**细节③：「我的电脑」文件**
如果玩家双击桌面上「我的电脑」图标——
里面只有一个C盘，一个D盘，一个G盘（名称：`MEMORY`）。
G盘打不开（提示：「请插入磁盘」）。

**细节④：回收站**
回收站里有一个文件：`不需要了.txt`
内容：
```
我的名字。
我的脸。
26年。
```
（恢复此文件无效，提示「该文件已被永久清除」）
