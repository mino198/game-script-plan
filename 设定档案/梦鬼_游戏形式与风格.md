# 《梦鬼》游戏形式与风格设定
> 版本：v0.1 | 最后更新：2026-06-19

---

## 一、游戏类型定位

**核心类型**：Galgame / 视觉小说（Visual Novel）
**子类型标签**：Horror · Mystery · Meta · Psychological

---

## 二、Meta结构设计

游戏存在**两个叙事层**：

### 外层（2026年·玩家层）
- 玩家以「现代人」身份启动游戏
- 游戏界面完全模拟 **Windows 98 桌面环境**
- 开场：玩家在二手交易平台买到一台旧电脑，开机发现桌面上有一个名为「梦鬼.exe」的文件
- 外层的「交互」包括：浏览桌面文件、读取日记文本文件、查看残存截图、尝试打开损坏的存档

### 内层（2000年·游戏层）
- 点击「梦鬼.exe」后进入内层，是一款画风老旧的Galgame
- 玩家操控「林智子」视角，经历五人互杀
- 内层有正常的Galgame交互：对话选项、好感度系统、多结局路径

### Meta穿透演出
随着游戏进行，外层与内层的边界开始模糊：
- 内层某些对话框会「渗出」到外层桌面上作为系统通知
- 外层桌面上的文件内容会随内层进度「更新」
- 某个时刻内层游戏「崩溃」，崩溃画面显示了不该出现的内容
- 最终内层的镜子演出穿透到外层，玩家（现实视角）拿起桌上的镜子

## 三、千禧年梦核美学（Dreamcore/Weirdcore）

### 核心情绪关键词
**疏离感 · 似曾相识 · 时间错位 · 熟悉的陌生 · 不安的温柔**

### 视觉风格参考
| 元素 | 处理方式 |
|------|---------|
| 色调 | 低饱和度暖黄/灰蓝为主，偶有过曝的白 |
| 光源 | 室内日光灯、老式显示器蓝光、窗外模糊的天空 |
| 纹理 | 扫描线噪点、轻微色差（chromatic aberration）、VHS质感 |
| 背景 | 空教室、走廊尽头、无人的操场、深夜便利店、老式游戏厅 |
| 字体 | 内层使用像素字体；外层使用Win98系统字体（宋体/新细明体） |
| 音效 | 老式电脑开机声、磁盘读取声、低频环境噪声、失真的BGM |

### 千禧年时代质感
- 内层游戏截图有明显的**JPEG压缩痕迹**
- 存档文件命名格式：`SAVE_001.DAT`
- 日记是`.txt`格式，无格式，偶有乱码
- 游戏内的「电话铃声」是老式翻盖手机的声音
- 背景出现的海报、杂志封面都是2000年前后的设计风格

## 四、Galgame结构规划

### 外层流程（线性）
```
开机 → 发现梦鬼.exe → 浏览碎片文件 → 启动游戏
→ [循环：玩内层 + 外层新文件解锁] → Meta穿透演出 → 结局
```

### 内层结构（分支）
```
序章：五人聚首，规则公布
├─ Chapter 1：第一轮副本（信任建立期）
├─ Chapter 2：第一个死亡（裂缝出现）
├─ Chapter 3：同盟与背叛
├─ Chapter 4：第五人消失
├─ Chapter 5：智子独自面对最终副本
└─ 结局A/B/C（根据选择分支）
```

### 结局方向（草案）
| 结局 | 触发条件 | 内容 |
|------|---------|------|
| 结局A（真结局） | 完成所有反转节点 | 智子记起愿望，游戏通关，梦鬼关闭，镜中反转 |
| 结局B（坏结局） | 错过关键节点 | 玩家再次被清洗记忆，成为下一个26年的宿主 |
| 结局C（隐藏结局） | 特定操作 | 玩家拒绝进入游戏，但梦鬼已经进入现实 |
